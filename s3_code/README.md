# EAG v3 - Session 3 Code

**Code from Session 3: Developer Foundations & Your First Agent**

This directory contains the code examples and exercises for Session 3 of the EAG (Enterprise Agent Guild) v3 Curriculum. The main objective of this session is to build Python and Node.js skills, and then create your first goal-directed agent with a working web UI.

## Contents

The directory includes various scripts to get you up and running with foundational concepts like asynchronous execution, basic language model integration, debugging, and ultimately building a fully functional agent:

- `01_code_interact_basic.py` - Basic interactive execution.
- `02_code_interact_agent.py` - Agentic code interaction basics.
- `03_pdb_basic.py` - Exploring the Python Debugger (PDB).
- `04_async_blocking.py` - Understanding blocking code.
- `05_async_nonblocking.py` - Understanding non-blocking asynchronous execution.
- `06_async_common_mistake.py` - Common pitfalls and mistakes with `asyncio`.
- `07_python_essentials.py` - Core Python fundamentals required for agent building.
- `08_llm_basic.py` - Baseline Large Language Model integration.
- `09_llm_with_system_prompt.py` - Interacting with an LLM using customized System Prompts.
- `10_full_agent.py` - Completing your first goal-directed agent.
- `11_fake_agent.py` - Mocking agent behaviors for testing and UI demonstration.
- `12_full_agent_ollama.py` - Implementing your agent locally utilizing Ollama models.

## Full Course Syllabus

Below is the complete EAG v3 syllabus spanning from foundational model architecture to full agent platform integration.

* **Session 1: Foundations of Transformer Architecture** – Understand neural networks, attention mechanisms, and positional encoding that power modern AI.
* **Session 2: Modern LLM Internals & The 2026 Model Landscape** – Learn tokenization, scaling laws, RLHF alignment, and the current state of reasoning and multi-modal models.
* **Session 3: Developer Foundations & Your First Agent** – Build Python and Node.js skills, then create your first goal-directed agent with a working web UI.
* **Session 4: MCP — The Tool Protocol** – Master the Model Context Protocol for tool registration, discovery, and invocation across servers.
* **Session 5: Planning, Reasoning & Structured Prompting** – Implement Chain-of-Thought, ReACT patterns, and self-validating task decomposition for intelligent agents.
* **Session 6: Cognitive Architecture & Adaptive Planning** – Design the 4-layer cognitive pipeline (Perception → Memory → Decision → Action) with strategy profiles and adaptive retry.
* **Session 7: Memory Systems & Modern RAG** – Build 3-tier memory (preferences, episodic, factual) with hybrid retrieval and semantic chunking.
* **Session 8: Multi-Agent Systems & DAG Architecture** – Coordinate multiple agents using directed acyclic graphs with parallel execution and session persistence.
* **Session 9: Browser Agents & Autonomous Web** – Automate web browsing with Playwright, vision-capable navigation, and multi-source research pipelines.
* **Session 10: Computer Use & Desktop Agents** – Control desktop applications using screen understanding, accessibility trees, and OS-level automation.
* **Session 11: Channel Architecture, Voice & Gateway** – Connect agents to WhatsApp, Slack, Discord, voice, and 20+ channels through a unified adapter pattern.
* **Session 12: Error Correction, Safety & Container Isolation** – Implement circuit breakers, JSON repair, and Docker-based sandboxing for safe agent execution.
* **Session 13: A2A — Agent-to-Agent Protocol** – Enable cross-vendor agent collaboration using Google's Agent2Agent protocol with capability discovery and task delegation.
* **Session 14: A2UI / AG-UI — Agent-to-User Interface** – Build agents that generate dynamic, interactive UIs at runtime using declarative and event-based protocols.
* **Session 15: Model Routing, Agent Economics & Observability** – Implement intelligent multi-model routing with cost tracking, budget controls, and OpenTelemetry instrumentation.
* **Session 16: Event-Driven Autonomous Agents** – Shift from reactive to proactive agents that monitor event streams, evaluate relevance, and act autonomously.
* **Session 17: Agentic Coding & Markdown-as-Code Skills** – Build coding agents with System 2 reasoning, codebase navigation, and markdown-driven skill injection.
* **Session 18: Agent Evaluation, Benchmarking & Capstone Prep** – Design custom eval harnesses, run GAIA/SWE-bench benchmarks, and prepare capstone proposals.
* **Session 19: Arcturus 2.0 — Full Integration & The Complete Platform** – Integrate all three protocol layers (MCP + A2A + A2UI) into a production-ready agentic platform.
* **Session 20: Capstone Pitches & Arcturus 2.0 Contributions** – Present 5-minute team pitches, review GitHub Projects, and kick off the 4-week capstone execution window.
